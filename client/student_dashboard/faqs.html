When do I update my profile:

When I log in for the first time as Student
When I get my Class results
When I get new certification
When I complete any project
When there is any change in my personal details

How do I update my profile:

Login as Student
Click on Profile
Enter Details in each segment
Make sure you have filled in your UG, PG and Current Qualification
Whenever you get result of any Class, you have to update the results
Click on Submit

My name does not appear in the list of my TPO:

Check if you have entered your Current Institute Name correctly
Check if you have entered your Current Degree Name correctly
Check if you have entered your UG/PG Name correctly
Check if you have entered your Current Year of Passing correctly
Check if your TPO has approved you

When I click on Register For Company, it shows I am not eligible:

Check if you have entered your Current Institute Name correctly
Check if you have entered your Current Degree Name correctly
Check if you have entered your UG/PG Name correctly
Check if you have entered your Current Year of Passing correctly
Check if your TPO has approved you
